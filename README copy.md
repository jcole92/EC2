# EC2 instance launch
EC2 instance launch and stop

## Overview
This project demonstrates the deployment a Windows EC2 instance within AWS. The goal of this project was to practice startting an instance and stopping one

---

## AWS Services Used
- EC2
- Security Groups
- IAM
---

## Project Objectives
- Launch a virtual server in AWS
- Understand the paramenters of EC2 features

---

## Steps Performed
1. Created an AWS EC2 instance
2. Tested connectivity and access controls
3.Stopped and terminated the Instance
